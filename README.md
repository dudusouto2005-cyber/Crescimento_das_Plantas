# Crescimento_das_Plantas


Foram avaliadas cinco plantas representativas da área experimental de 42 m².

A altura média observada foi de aproximadamente 44 cm, indicando crescimento uniforme entre as plantas.

Essa uniformidade demonstra que o manejo adotado durante o cultivo, incluindo irrigação e adubação, proporcionou condições favoráveis ao desenvolvimento inicial da cultura.

O acompanhamento da altura das plantas permite identificar precocemente possíveis problemas de manejo e auxilia na tomada de decisões durante o ciclo produtivo.
